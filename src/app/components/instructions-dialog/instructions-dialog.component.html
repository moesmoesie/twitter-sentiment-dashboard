Instructions

  • To populate the dashboard, you must add keywords to the keyword input field.
  • Entering "#Helloworld" as a keyword will show tweets that contain the hashtag "#Helloworld"
  • Entering "@BarackObama" will include that mention the user "@BarackObama".
  • Entering "from:BarackObama" will include tweets made by the user "@BarackObama"
  • Entering "Hello World" will include tweets that contain the text "Hello World"
